# 🧠 Replit Dev Rules

1. Top priority: Fetch fast, accurate sports data
2. OpenAI + weather integration = critical
3. No random UI changes
4. Always deduplicate alerts
5. Ask before deploying untested changes
6. **ALL SPORTS ALERTS MUST BE KID-FRIENDLY AND CLEAR** - No technical jargon, codes, or confusing abbreviations. Explain what's happening in simple language that anyone can understand.
