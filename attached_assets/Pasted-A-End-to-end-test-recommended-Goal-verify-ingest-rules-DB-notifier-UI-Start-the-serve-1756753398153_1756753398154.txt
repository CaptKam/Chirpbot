B) Inject a synthetic alert (fastest UI check)

Goal: push a single fake alert through the same path your UI listens to.

1) Add a tiny admin route

server/http/test-alert.ts

import express from 'express';
import { buildAlertKey } from '../models/alert-key';
import type { AlertCandidate } from '../models/contracts';
import { tryInsertAlert, setCooldown } from '../services/db';
import { pushAlert } from '../services/notifier';

export const testAlert = express.Router();

// GET /admin/test-alert?type=HIGH_SCORING_OPP&sport=MLB&gameId=DEMO1
testAlert.get('/admin/test-alert', async (req, res) => {
  const { type='HIGH_SCORING_OPP', sport='MLB', gameId='DEMO1' } = req.query as Record<string,string>;

  const alert: AlertCandidate = {
    sport: sport as any,
    gameId,
    type: type as any,
    phase: 'TEST_PHASE',
    situation: `TEST_${Date.now()}`,   // unique to bypass dedup on repeats
    ruleVersion: 'ruleset_test',
    score: 77,
    context: { note: 'Synthetic test alert' }
  };

  const inserted = await tryInsertAlert(alert);
  if (!inserted) return res.status(200).json({ ok: true, note: 'duplicate-key (expected if you re-use situation)' });

  await pushAlert(alert);     // goes to Telegram/WS/webhook per your notifier
  await setCooldown(alert);   // optional: behave like real alerts

  res.json({ ok: true, alert_key: buildAlertKey(alert), alert });
});


Wire it in your server bootstrap (e.g., server/index.ts):

import express from 'express';
import { health } from './http/health';
import { testAlert } from './http/test-alert';

const app = express();
app.use(health);
app.use(testAlert);
// ... existing routes / ws setup

C) WebSocket-only smoke (bypass DB/rules)

Goal: prove the page reacts to live pushes even without the rest of the stack.

1) Server broadcast helper

Wherever you keep your WS server, add a tiny helper:

// server/services/ws-bus.ts
import type { WebSocket } from 'ws';
export const wsClients = new Set<WebSocket>();
export function broadcastAlert(payload: any) {
  const msg = JSON.stringify({ type: 'alert', payload });
  for (const ws of wsClients) {
    if ((ws as any).isAlive !== false) ws.send(msg);
  }
}


Hook connection (e.g., in your ws setup):

wss.on('connection', (ws) => {
  (ws as any).isAlive = true;
  ws.on('pong', function heartbeat(){ (this as any).isAlive = true; });
  wsClients.add(ws);
  ws.on('close', () => wsClients.delete(ws));
});

Polling fallback (optional, handy while debugging):

// GET /alerts?since=ISO_5min_ago&limit=50
