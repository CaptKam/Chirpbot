# 🎨 ChirpBot V2 UI Theme Guide

## Theme Style
Modern, bold, and data-forward with a professional sports app aesthetic.

## Visual Tone
- Dynamic & high-energy
- Clean & modular
- User-first layout

## Design System
- Typography: Heavy uppercase titles, sans-serif body text
- Color Palette:
  - Primary: Deep Blue (#1D2E5F)
  - Accent: Scarlet Red (#E23237)
  - Background: Light Gray (#F3F4F6)
  - Contrast: White (#FFFFFF), Dark Gray (#4A4A4A)
- Buttons: Rounded pill-style, high contrast
- Cards: Rounded, layered, with shadows
- Navigation: Bottom nav with bold icons
- Live Indicators: Red pill tags with bold text
- Imagery: Hero players, dynamic sports photos
- Logos: Flat vectors, left vs. right display

## Components
- Intro: CTA and large image
- Home: Live alerts + upcoming matchups
- Matches: Sortable by league/date
- Teams: Schedule, stats tabs
- Results: Breakdown with alert summary
