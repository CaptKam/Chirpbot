# 📱 ChirpBot V2 Page Layout

## 1. Calendar Page
- Tabs by sport (MLB, NFL, NBA, etc.)
- List of teams with monitor toggles
- Optional: game status, weather tags

## 2. Alerts Page
- Stream of live and past alerts
- Filters: sport, type, AI confidence
- Alert card: time, game, OpenAI context

## 3. Settings Page
- Tabs by sport
- Alert logic toggles (e.g., Red Zone, RISP)
- AI filter toggle + confidence slider

## Navigation
- Bottom nav bar: Calendar | Alerts | Settings
