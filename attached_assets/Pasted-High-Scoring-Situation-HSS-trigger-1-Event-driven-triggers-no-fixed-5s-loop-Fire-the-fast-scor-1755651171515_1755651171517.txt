High‑Scoring Situation (HSS) trigger
1) Event‑driven triggers (no fixed 5s loop)

Fire the fast scorer whenever any of these diffs hit:

Base/outs change (hit, walk, SB/CS, WP/PB)

Batter/on‑deck changes (new PA starts; on‑deck updates)

Count crosses to 2–0 or 3–1

Pitcher or defensive switch

Weather delta ≥ 5 mph OR wind vector rotates ≥ 45°

Inning/leverage crosses: 7th, 9th, LI ≥ 1.5/2.0

If the fast score passes threshold → call the LLM engine.

2) Fast HSS score (0–1) — cheap, deterministic

Use a weighted score; only call the LLM when score ≥ threshold_by_state.

Features & weights (tuneable):

Base–Out State (0.00–1.00, w=0.30)
Pre-fill a small table (proxy for RE24):

2nd&3rd, 0 outs → 1.00

2nd&3rd, 1 out → 0.85

3rd, 1 out → 0.70

Bases loaded, 1 out → 0.90

Others scale down accordingly

Leverage Index LI (0–1, w=0.15) → clamp(LI / 3.0, 0, 1)

Score Tightness (0–1, w=0.10) → 1 if |diff| ≤ 1, 0.7 if =2, else 0.2

Batter Power (0–1, w=0.12) → clamp(ISO / 0.25, 0, 1) or 1 if HR≥25

On‑Deck Power (0–1, w=0.08) → 1 if HR≥20 else clamp(next_ISO/0.22,0,1)

Platoon Edge (0–1, w=0.05) → 1 on advantage, 0.5 neutral, 0 disadvantage

Pitcher Fatigue (0–1, w=0.05) → 1 if TTO≥3 or pitch#>75 (RP>25)

Bullpen Quality Opp (0–1, w=0.05) → 1–ERA_relief/5 (clamped 0–1)

Wind Toward OF (0–1, w=0.07) → if vector≈CF: clamp(mph / 12, 0, 1)

Park HR Factor (0–1, w=0.03) → clamp(park_HR / 110, 0.5–1)

Thresholds by context:

Cold: 0.70

Warm: 0.55

Hot/Critical (7th+, LI≥2, tie/±1): 0.40

Worked example (your case)

Runners 2nd & 3rd, 1 out → 0.85 × 0.30 = 0.255

Wind out to CF 10 mph → (10/12≈0.83) × 0.07 = 0.058

On‑deck HR ≥ 20 → 1 × 0.08 = 0.080

Assume LI 1.6 (late and close) → (1.6/3≈0.53) × 0.15 = 0.080

Score diff 0–1 → 1 × 0.10 = 0.100

Batter power decent (ISO .220) → (.22/.25=0.88) × 0.12 = 0.106

Others middling (say ~0.10 total)

Total ≈ 0.255 + 0.058 + 0.080 + 0.080 + 0.100 + 0.106 + 0.10 = 0.779
→ Exceeds any threshold → Run the AI engine now.

3) Deduping & cooldowns

If LLM runs and no alert fires, set recheck TTL (unless a major delta occurs):

Warm: 25s, Hot: 10s, Critical: 6s

If an alert fires for the same base–out + batter context, suppress duplicates for 120s unless score improves by ≥ 15% (e.g., wind jumps, count becomes 3–1, or new power batter).

4) Minimal code (TypeScript‑ish)
type GameDelta =
  | "basepath" | "outs" | "batter" | "onDeck" | "count_cross_2-0" | "count_cross_3-1"
  | "pitcher_change" | "def_change" | "weather_delta" | "inning_cross_7" | "li_cross";

function hssScore(ctx: Ctx): number {
  const baseOut = tableBaseOut(ctx);       // returns 0..1 (e.g., 0.85 for 2nd&3rd,1 out)
  const li = clamp(ctx.li / 3, 0, 1);
  const scoreTight = ctx.scoreTight;       // 1, .7, .2
  const batterPower = Math.max(ctx.batter.hr>=25?1:0, clamp(ctx.batter.iso/0.25,0,1));
  const onDeckPower = ctx.onDeck.hr>=20?1:clamp(ctx.onDeck.iso/0.22,0,1);
  const platoon = ctx.platoonEdge;         // 0, .5, 1
  const fatigue = ctx.pitcherFatigue;      // 0..1
  const penQual = clamp(1 - ctx.oppBullpenERA/5, 0, 1);
  const windCF = ctx.wind.toCF ? clamp(ctx.wind.mph/12, 0, 1) : 0;
  const park = clamp(ctx.park.hrFactor/110, 0.5, 1);

  return 0.30*baseOut + 0.15*li + 0.10*scoreTight + 0.12*batterPower +
         0.08*onDeckPower + 0.05*platoon + 0.05*fatigue + 0.05*penQual +
         0.07*windCF + 0.03*park;
}

function shouldRunLLM(ctx: Ctx, deltas: GameDelta[]): boolean {
  const score = hssScore(ctx);
  const tier = ctx.li>=2 || (ctx.inning>=7 && Math.abs(ctx.lead)<=1) ? "hot" :
               (ctx.hasRISP || Math.abs(ctx.lead)<=2 ? "warm" : "cold");
  const threshold = tier==="hot" ? 0.40 : tier==="warm" ? 0.55 : 0.70;

  const majorDelta = deltas.some(d => ["basepath","batter","onDeck","pitcher_change","weather_delta"].includes(d));
  const pastTTL = Date.now() >= ctx.nextAllowedEvalAt;

  return (score >= threshold && pastTTL) || (majorDelta && pastTTL);
}

// on any game update:
function onGameUpdate(gameId: string, diff: Diff) {
  const ctx = state[gameId].ctx = buildCtx(gameId, diff);
  const deltas = identifyDeltas(diff);
  if (shouldRunLLM(ctx, deltas) && budgetOk()) {
    runLLM(ctx);
    state[gameId].nextAllowedEvalAt = Date.now() + ttlByTier(ctx); // e.g., 10s in hot
  }
}
