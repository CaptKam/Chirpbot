Key research takeaways

Direction matters more than sheer speed – MLB’s own analysis shows that just 5 mph of tailwind can add nearly 19 feet of carry to a ball
mlb.com
. More importantly, moderate winds blowing out to left field produce more runs and homers than even much stronger winds to center or right field
rotogrinders.com
; winds to right field tend to suppress homers as the ball is pushed toward deeper center
rotogrinders.com
.

Winds blowing out vs. in – At Wrigley Field, a 10 mph wind blowing out increases home runs by ~50%, while the same speed blowing in reduces home runs by ~33%
rotogrinders.com
. This suggests that a 10 mph directional shift is highly significant.

Cross-winds can have opposite effects depending on direction; right-to-left winds boost scoring, while left-to-right winds suppress it

.

Suggested code changes

Lower the speed-change threshold: Since 5 mph can add ~19 feet of distance
mlb.com
, use a 5 mph threshold instead of 6 mph.

private readonly MIN_SPEED_CHANGE = 5; // mph


Refine direction impact values: Give winds to left the highest positive impact, winds to center/out moderate benefit, winds to right a smaller benefit, winds blowing in a negative impact, and treat cross winds as neutral if the orientation isn’t specified. This more closely reflects the research showing left-field winds are most favorable
rotogrinders.com
 and right-field winds least
rotogrinders.com
.

private getDirectionImpact(direction: string): number {
  if (!direction || typeof direction !== 'string') return 0;
  const dir = direction.toLowerCase();
  if (dir.includes('left')) return 3;          // most favorable (out to left)
  if (dir.includes('out') || dir.includes('center')) return 2; // moderate
  if (dir.includes('right')) return 1;         // least favorable of “out”
  if (dir.includes('in')) return -2;           // blowing in
  return 0;                                     // cross/unknown
}


Detect smaller direction shifts: Change hasSignificantDirectionChange to trigger when the impact category differs by 1 or more, rather than 2. Subtle changes (e.g., from left to center) can still alter scoring
rotogrinders.com
.

private hasSignificantDirectionChange(oldDir: string, newDir: string): boolean {
  const oldImpact = this.getDirectionImpact(oldDir);
  const newImpact = this.getDirectionImpact(newDir);
  return Math.abs(oldImpact - newImpact) >= 1;
}


Require significant speed or direction change before alerting: Instead of alerting on any change, trigger only when the speed difference meets MIN_SPEED_CHANGE or the direction change is significant. This avoids spamming alerts for trivial variations.

// Calculate changes
const speedChange = Math.abs(currentWind.windSpeed - previous.speed);
const directionChanged = this.hasSignificantDirectionChange(
  previous.direction,
  currentWind.windDirection
);
// Only alert if change exceeds threshold
if (speedChange >= this.MIN_SPEED_CHANGE || directionChanged) {
  // ...
}
