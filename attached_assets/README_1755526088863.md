# ChirpBot V2 (Replit Starter Packet)

This is your new app structure based on updated specs:
- /calendar page to monitor teams
- /alerts page to view AI-enriched alerts
- /settings page to manage sport-specific alert rules

## Routes
- `/calendar`: Pick teams by sport
- `/alerts`: Filter and view live alerts with OpenAI context
- `/settings`: Choose high-scoring scenarios per sport

## How to Start
1. Update `.env` with your Telegram credentials
2. Run the app
