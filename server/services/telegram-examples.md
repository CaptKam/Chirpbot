## Key Improvements
1. **Single emoji per sport** - Clean visual identity
2. **No duplicate information** - Teams appear only once
3. **3-line structure** - Title, Score, Situation
4. **Context only when critical** - Runners/down & distance only for relevant alerts
5. **Consistent hashtags** - #[Sport]Alert #ChirpBot
6. **Clean action link** - Simple "View Alert Details"
