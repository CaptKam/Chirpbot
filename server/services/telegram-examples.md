
# Telegram Alert Examples - Universal Format

## V3 Design Principles
- **Single emoji per sport** - Clean visual identity
- **Consistent 5-line structure** - Title, Score, Situation, Link, Hashtag
- **3-second readability** - Essential info only
- **No duplicate information** - Each piece of data appears once

## Key Improvements
1. **Consistent Structure** - Every alert follows the same 5-line format
2. **Sport-Specific Context** - Only relevant game situation details shown
3. **Clean Typography** - Single emoji, clear separators, no redundancy
4. **Universal Readability** - 3-second scan reveals all essential information
5. **Scalable Design** - Works for any sport with minimal customization
