## 🚀 Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure environment:**
   ```bash
   cp .env.example .env
   # Add your DATABASE_URL and API keys
   ```

3. **Initialize database:**
   ```bash
   npm run db:push
   ```

4. **Start development:**
   ```bash
   npm run dev
   ```

5. **Access application:**
   - Web: http://localhost:5000
   - Demo: username "demo"
